Carregar uma shapefile

Em vez de desenhar a área que pretende confrontar com os instrumentos de gestão do território, pode carregar uma shapefile previamente existente. Use o botão Carregar shapefile para o efeito.

Uma shapefile é constituída por vários ficheiros. Por isso, antes de carregar uma shapefile, tem que criar um zip com os vários ficheiros que compõem a shapefile. Envie o zip.

Exemplo

Preparámos um zip com uma shapefile com um terreno desenhado. Serve como exemplo de um zip que poderá ser submetido na aplicação. É o ficheiro edificio001.zip.

Sistemas de coordenadas

Tem que existir um ficheiro com o sistema de coordenadas, para que a shapefile possa ser corretamente importada. Pode enviar a shapefile em qualquer sistema de coordenadas usual (EPSG:4326, EPSG:27492, EPSG:3763, etc).

No exemplo, existe um ficheiro edificio001.prj, que define o sistema de coordenadas.
