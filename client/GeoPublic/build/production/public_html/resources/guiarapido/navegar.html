Mapa

O mapa serve de fundo a toda a interação. Para trabalhar sobre o mapa, escolha o painel Mapa.

Navegar

Use o rato para navegar no mapa. Clique e arraste o mapa de um lado para o outro (sem largar o botão do rato).

Para ampliar ou reduzir o mapa, use a rodinha do rato.

Para ampliar uma determinada área, carregue em Shift e clique e arraste o rato para desenhar a área que lhe interessa focar.

Mapa de fundo

No início, aparece-lhe o mapa OpenStreetMap. Pode escolher outro mapa de fundo. Use o (+) no canto superior direito do mapa para o efeito.
