Introdução

Basta-lhe 2 ou 3 minutos para percorrer este Guia Rápido para ficar a conhecer as principais funcionalidades da Informação Prévia no minuto.
Este Guia está dividido em várias páginas, sendo cada uma delas focada num único aspeto.
Vá avançando as páginas com o botão Seguinte. Com o botão Anterior pode rever páginas anteriores.
Este Guia está sempre disponível para consulta no painel Bem vindo.
